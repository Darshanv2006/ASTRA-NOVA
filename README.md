## ✨ Features

### Core Capabilities

* **Real-time 3D Earth Visualization** - Interactive globe rendering satellites and debris
* **Satellite Fleet Tracking** - Monitor thousands of satellites
* **Collision Detection & Alerts** - AI-powered risk assessment
* **Debris Monitoring** - Track orbital debris
* **Mission Planning** - Simulate satellite maneuvers
* **Live Telemetry Dashboard** - Real-time health metrics
* **Analytics & Reporting** - Generate reports

## 🚀 Installation & Setup

```bash
# Clone the repository
git clone https://github.com/Darshanv2006/ASTRA-NOVA.git

# Navigate into project folder
cd ASTRA-NOVA

# Install dependencies
npm install --legacy-peer-deps

# Run frontend
npm run dev
```

👉 Open: http://localhost:5173
