## 🌌 Overview  
ASTRA-NOVA is a next-generation space intelligence platform designed to monitor satellite health, predict orbital collisions, and visualize space activity in real-time.

It combines **AI-driven analytics**, **orbital mechanics**, and **modern web technologies** to build a powerful simulation system.

---

## ✨ Features  

### 🌍 Earth Visualization
- Interactive 3D globe with real-time rotation  
- Satellite orbit visualization  
- Space debris tracking  
- Atmospheric effects  

### 🛰️ Satellite Fleet Management
- Real-time tracking using SGP4  
- TLE data integration  
- Fleet overview dashboard  
- Detailed telemetry  

### 🚨 Collision Detection
- AI-powered anomaly detection  
- Real-time proximity alerts  
- Risk classification system  
- 3D distance analysis  

### 📡 Mission Control
- Mission planning system  
- Timeline visualization  
- Status tracking  
- PDF reports  

### 📊 Analytics
- System health monitoring  
- Telemetry history  
- Alert statistics
